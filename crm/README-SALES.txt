Devox Sales — Setup Instructions
=================================

Welcome! Follow these steps to install the Devox Sales app on your PC.


REQUIREMENTS
------------
Windows 10 or 11
Internet connection
Python 3.10 or newer
  (Download from: https://www.python.org/downloads/)
  IMPORTANT: When installing Python, check
  "Add Python to PATH" on the first screen.


INSTALLATION (3 steps)
----------------------
1. Unzip this folder anywhere (Desktop or Documents).

2. Double-click on  install.bat

3. The installer will:
     - install required packages
     - create a "Devox Sales" shortcut on your Desktop
     - launch the app

That's it. From now on you can launch the app
by double-clicking "Devox Sales" on your Desktop.


LOGIN
-----
Use the username and password your administrator
gave you. If you forgot your password, ask your
admin to reset it from the Admin tab.


TROUBLESHOOTING
---------------
- Python not found?
  Re-install Python from python.org and check
  "Add Python to PATH". Then run install.bat again.

- Cannot connect to server?
  Make sure you have internet. The CRM URL is
  inside crm_url.txt — your admin gave you this file.

- Other issues?
  Contact your admin.
